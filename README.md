# road-defect-detection-yolov8
Проект реализует пайплайн детекции дефектов дорожного полотна на датасете RDD2022. Используется модель YOLOv8и кастомные аугментации через Albumentations. В работе проведена EDA, отбор аугментаций, сравнение оптимизаторов и конфигураций; достигнуты приемлемые значения mAP@50 и FPS для задач мониторинга в реальном времени.
